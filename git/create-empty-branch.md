### Clone Repository

```shell
git clone https://github.com/user/repo.git
```

### Open Directory

```shell
cd repo
```

### Create New Branch

```shell
git switch --orphan my-new-branch
```

### Create Empty Commit

```shell
git commit --allow-empty --allow-empty-message --no-edit
```

### Publish Branch

```shell
git push --set-upstream origin my-new-branch
```
